python时间处理库

pip install python-dateutil

在mongodb中插入日期类型

import pymongo
from dateutil import parser
dateStr = "2019-05-14 01:11:11"
myDatetime = parser.parse(dateStr)
client = pymongo.MongoClient(host="127.0.0.1", port=27017)
db = client["test"]
db.ceshi.insert({"date": myDatetime})
client.close()

在mongodb中查询大于/小于某个时间点的数据

documents = collection.find({"date": {"$lt": datetime(2019, 1,1,15,40,3)}}).sort('date')

python 打包pandas文件

  1. 首先第一次进行打包，会在py目录下生成spec文件，是一个打包脚本

  2. 编辑其内容。添加隐藏的包

    hiddenimports=[ 'pandas', 'numpy' ]

  3. 如果遇到 recursion limit 错误, 在spec文件中添加如下两行

import sys
sys.setrecursionlimit(2000000
